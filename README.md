# MonCar - Car Rental System

Fullstackowy akademicki projekt systemu wypożyczalni samochodowej zrealizowany w PHP i MySQL. Aplikacja umożliwia przeglądanie floty, zarządzanie opiniami oraz posiada system autoryzacji użytkowników.

## Funkcjonalności

* **System kont**: Rejestracja i logowanie użytkowników (bezpieczne hashowanie haseł `password_hash`)
* **Dynamiczna flota**: Pobieranie danych o samochodach z bazy danych MySQL
* **Galeria**: Podgląd zdjęć samochodów
* **Opinie**: Możliwość dodawania i przeglądania ocen dodawanych przez zalogowanych użytkowników
* **Kontakt**: Formularz kontaktowy przesyłający wiadomości do bazy
* **Licznik odwiedzin**: System śledzenia popularności witryny

## Stack

* **Backend:** PHP
* **Database:** MySQL
* **Frontend:** HTML, CSS, Bootstrap, JavaScript
* **Serwer:** XAMPP / Apache
